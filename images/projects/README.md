# Project Images

Place your project thumbnail images in this folder.

## File Naming Convention

Each image must match the `slug` in `projects/data.js`:

| Image File                     | Slug                   |
|-------------------------------|------------------------|
| `autoprompt-sam.jpg`          | autoprompt-sam         |
| `iot-smart-conveyor.jpg`      | iot-smart-conveyor     |
| `or-optimization.jpg`         | or-optimization        |
| `shoe-cleaning-machine.jpg`   | shoe-cleaning-machine  |

## Supported Formats
- `.jpg` / `.jpeg` (recommended, smaller file size)
- `.png`
- `.webp` (best for web performance)

## Recommended Size
- **Width:** 800px minimum
- **Aspect ratio:** 16:9 or wider landscape (the card crops to a fixed 168px height)

## Fallback
If an image is missing or fails to load, the card automatically shows a Lucide icon placeholder instead.
